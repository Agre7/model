    Public API

    Model()

    • The model constructor function.
    • Using “new” is optional.
    • Examples: var model = Model(), var model = new Model()
    • The returned model object can be treated as a plain JavaScript Object for setting and getting property values.
    • Examples: model.x = 5; console.log(model.x)

    Returns a function which when called schedules the original callback function to execute on the next tick of the JavaScript event loop. Multiple calls to the debounced function within a single tick of the event loop cause the original callback to be called only once.

      function debounce(callback){
    var queued = false;
    return function () {
      if(!queued){
        queued = true;
        setTimeout(function () {
          queued = false;
          callback();
        }, 0);
      }
    };
  }
